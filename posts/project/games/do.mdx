## Features

- Arcade
- Engine used: Unity
- Developed for a 1-night-2-day game development competition

## How to Play

- Prepare the next platform before the rolling ball falls.
- Game ends if the ball falls.
- Bonus awarded for setting up the next platform at the same height as the previous one.
